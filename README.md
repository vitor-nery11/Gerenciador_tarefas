# 📝 Gerenciador de Lista de Tarefas (Python)

Um simples **gerenciador de tarefas no terminal**, desenvolvido em Python, que permite adicionar, visualizar, atualizar, completar e remover tarefas.

Esse projeto foi criado com o objetivo de **praticar conceitos básicos de programação em Python**, como funções, listas, dicionários, loops e interação com o usuário.

---

# 🚀 Funcionalidades

O programa possui um menu interativo no terminal com as seguintes opções:

1. **Adicionar tarefa**  
   Permite adicionar uma nova tarefa à lista.

2. **Ver tarefas**  
   Mostra todas as tarefas cadastradas com seu status.

3. **Atualizar tarefa**  
   Permite alterar o nome de uma tarefa existente.

4. **Completar tarefa**  
   Marca uma tarefa como concluída.

5. **Deletar tarefas completadas**  
   Remove todas as tarefas que já foram concluídas.

6. **Sair**  
   Finaliza o programa.

---

# 🧠 Conceitos praticados

Este projeto utiliza vários conceitos importantes de Python:

- Funções
- Listas
- Dicionários
- Estrutura de repetição (`while`, `for`)
- Estruturas condicionais (`if`, `elif`)
- Manipulação de dados
- Entrada de dados com `input()`
- Enumeração com `enumerate()`
